Text Editor 1.0 Help

In this app, the user need to create an account before using the app. There are 2 types of account:

  • View Only: User can only open and view the file
  • Edit: User has full permission to open, edit, and make changes to the file

The feature of Text Editor at this version includes the following with its shortcut:

  1. Basic Text Viewing: Open (Ctrl+O), Save (Ctrl+S), Save as (Ctrl+Shift+S), Copy (Ctrl + C), Cut (Ctrl+X), Paste (Ctrl + C)
  2. Text Manipulation: More than 150 Font Family to choose and several font sizes

There are 3 major parts of the app:

  1. Menu Strip:
    • File: New, Open, Sav, Save-as, Logout
    • Edit: Copy, Cut, Paste
    • Help: About the app version and author
  2. Top Tool Strip
    • Same as File in Menu Strip
    • Font Family (150+ Fonts) and Font size (any valid number)
  3. Left Tool Strip
    • Simple editing tools: Copy, Cut, Paste
